1.typeof operator?


Answer:


The typeof operator returns type information as a string that identifies the data type of an expression. There are six possible values that typeof returns: "number," "string," "boolean," "object," "function," and "undefined”.

3.operator precedence, explain by an example?


Answer:


Operator precedence determines the order in which operators are evaluated. Operators with higher precedence are evaluated first. Operator precedence describes the order in which operations are performed when an expression is evaluated. Operations with a higher precedence are performed before those with a lower precedence. For example, multiplication is performed before addition.

Examples:


“leela”+ 2+3 = leela23 //left to right


1+2+”leela”= 3leela //left to right


“hi”+(2+3)=hi5. // () will calculate first because it has highest priority.
